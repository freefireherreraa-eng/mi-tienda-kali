# mi-tienda-kali
Mi sitio web es un blog técnico sobre ciberseguridad y Kali Linux. Proporciono tutoriales de auditoría de redes inalámbricas y recomiendo el hardware compatible necesario para laboratorios de hacking ético, como adaptadores Wi-Fi específicos (RT3070), Raspberry Pi y libros de seguridad informática para estudiantes y profesionales del sector.
